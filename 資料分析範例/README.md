# 資料分析範例

本資料夾收錄使用 Python 進行資料分析的完整範例，內容涵蓋資料取得、清理、合併、分組統計、資料重塑、時間序列與視覺化。主要使用 pandas、NumPy、Matplotlib、Seaborn 及 pandas-datareader。

點選範例名稱即可查看程式與分析流程。部分範例需要網路連線或搭配 `datasets`、`examples` 目錄中的資料檔案。
