# 學生用 Prompt:用 AI 生成 Streamlit 營業額統計儀表板

## 使用說明(給學生)

把下面整段文字複製,貼給 AI(例如 Claude、ChatGPT),AI 就會幫你產生一支可執行的 Streamlit 程式。拿到程式後存成 `app.py`,在終端機輸入 `streamlit run app.py` 即可看到網頁儀表板。

---

## 複製以下這段給 AI ⬇️

```
請幫我用 Python 的 Streamlit 寫一支可執行的營業額統計儀表板程式,需求如下:

【資料】
- 用 numpy 亂數產生「8 家分店」在「週一到週日共 7 天」的每日營業額。
- 每天營業額是 8000 到 20000 元之間的整數。
- 用 pandas 建立 DataFrame,欄位是週一到週日,index 是分店編號 1~8。
- 為了讓每次重新整理結果一致,請用 np.random.seed 固定亂數種子。

【統計計算】
- 計算每家分店的「週總額」(7 天加總)。
- 計算每家分店的「日平均」。
- 依週總額由高到低,用 rank 算出「業績排名」(業績最高排第 1)。
- 把週總額、日平均、業績排名加成 DataFrame 的新欄位。

【Streamlit 介面】
- 最上方用 st.title 顯示標題「飲料連鎖店一週營業額儀表板」。
- 用 st.metric 顯示三個重點數字:全公司總營業額、單一分店最高週總額、整體日平均。
- 用 st.dataframe 顯示完整的統計表,並依業績排名排序。
- 加入一個 st.selectbox,讓使用者可以選擇要看「週總額」還是「日平均」,圖表會跟著切換。

【圖表整合】
- 用長條圖(bar chart)顯示 8 家分店的營業額(依上面 selectbox 的選擇),X 軸是分店編號,Y 軸是金額。
- 用折線圖(line chart)顯示「全公司每一天(週一到週日)的總營業額」變化趨勢。
- 圖表可以用 st.bar_chart、st.line_chart,或用 matplotlib / plotly 繪製皆可,請選一種並確保能正常顯示中文。

【其他要求】
- 程式要完整、可直接執行,不要省略任何部分。
- 在重要的程式行加上中文註解,方便我理解。
- 最後告訴我:需要先安裝哪些套件、以及啟動指令是什麼。
```

## 給老師的補充建議

這份 prompt 刻意涵蓋四個層次,方便你引導學生觀察 AI 的輸出品質:資料生成、統計計算、Streamlit 元件、圖表整合。可以延伸的教學討論點包括:中文字型在圖表中常會變成方框,讓學生試著請 AI「解決 matplotlib 中文顯示問題」;以及 `np.random.seed` 對「每次重整資料是否一致」的影響。
