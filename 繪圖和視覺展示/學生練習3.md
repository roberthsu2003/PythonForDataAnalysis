# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
import matplotlib.pyplot as plt
labels = ['Nokia', 'Samsung', 'Apple', 'Lumia']
values = [20, 30, 45, 10]
colors =['yellow', 'green', 'red', 'blue']

figure = plt.figure()
axes = figure.add_subplot()
axes.pie(values,labels=labels,colors=colors,explode=[0.3,0,0,0],startangle=180,shadow=True,autopct='%1.1f%%')
plt.show()
```

## 使用的 Python 技術

- `import matplotlib.pyplot as plt`
- list(串列):labels、values、colors、explode
- `plt.figure()` 與 `add_subplot()`
- `axes.pie()` 及其參數:labels、colors、explode、startangle、shadow、autopct
- `plt.show()`

## 學習目標

學生可以學會:

- 用串列準備圓餅圖的資料(數值、標籤、顏色)
- 呼叫 `pie()` 並設定常用參數
- 控制圓餅圖外觀:突出某一塊、設定起始角度、顯示百分比
