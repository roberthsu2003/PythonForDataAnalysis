# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
#subplot()可顯示1個figure 多個Axes
#第一個參數垂直分割
#第二個參數水平分割
#第三個選擇那一個

import numpy as np
import matplotlib.pyplot as plt

t = np.arange(0, 5, 0.1)
y1 = np.sin(2 * np.pi * t)
y2 = np.sin(2*np.pi*t)

plt.subplot(211)
plt.plot(t, y1, 'b-.')
plt.subplot(212)
plt.plot(t,y2,'r--')
```

## 使用的 Python 技術

- `import numpy as np`、`import matplotlib.pyplot as plt`
- `np.arange()` 建立數值陣列
- `np.sin()`、`np.pi` numpy 數學運算(向量化計算)
- `plt.subplot()` 切割版面(垂直、水平分割與選圖)
- `plt.plot()` 繪圖與線條樣式字串('b-.'、'r--')

## 學習目標

學生可以學會:

- 用 `np.arange()` 產生等間隔的數值序列
- 用 numpy 對整個陣列做數學運算(不必寫迴圈)
- 用 `subplot()` 在同一張圖上排列多個子圖,並控制線條樣式
